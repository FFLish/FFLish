## About FFLish

**FFLish** is a competitive **FIRST LEGO League (FLL)** robotics team and part of the **PinkAlliance**.

<table>
<tr>
<td>

• Robot programming and competition strategy  
• Python-focused development  
• Web development and database fundamentals  
• Qualified for **World Finals (South Africa)**  
• Currently competing in **Unearthed**  
• Based in **Heidelberg, Germany**

</td>
</tr>
</table>
